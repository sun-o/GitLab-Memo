##开发人员的工作流程

1. 克隆项目
```
git clone git@example.com:project-name.git
```

2. 创建特性分支
```
git checkout -b $feature_name
```

3. 编写代码并提交
```
git commit -am "My feature is ready"
```

4. 推送你的分支到 GitLab
```
git push origin $feature_name
```

5. 在 Commits Page 审查你提交的代码

6. 创建 Merge Request

7. 你的项目负责人将会审核代码, 然后合并到主分支
